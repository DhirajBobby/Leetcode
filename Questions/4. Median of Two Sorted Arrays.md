## Problem
#hard #binarysearch 
Given two sorted arrays `nums1` and `nums2` of size `m` and `n` respectively, return **the median** of the two sorted arrays.

The overall run time complexity should be `O(log (m+n))`.

**Example 1:**

`Input: nums1 = [1,3], nums2 = [2]`
`Output: 2.00000`
`Explanation: merged array = [1,2,3] and median is 2.`

**Example 2:**

`Input: nums1 = [1,2], nums2 = [3,4]`
`Output: 2.50000`
`Explanation: merged array = [1,2,3,4] and median is (2 + 3) / 2 = 2.5.`

## Solution

We try to create a left partition of total//2 elements. then the largest element of that will be our target. (or mean of largest of that and smallest of remaining) 

Let A be smaller array and B be larger.

We first take till mid of A and rest from B. Then check if left partition (Aleft + Bleft) is valid.
If it is valid we got the correct left partition. So now we calculate the median.

If the left partition is invalid, we change l and r pointer in A accordingly and find new mid for A.

``` python
  class Solution:
    def findMedianSortedArrays(self, nums1: List[int], nums2: List[int]) -> float:

        A,B = nums1, nums2
        # make sure A is the smaller array 
        if len(A) > len(B):
            A,B = B,A

        total = len(A) + len(B)
        k = total // 2

        l = 0
        r = len(A) - 1

        while True:
            ma = (l+r) // 2
            mb = k - ma -2
            
            # The values we need to compare inorder to check if partitioning is                       correct
            Aleft = A[ma] if ma >=0 else float("-infinity")
            Aright = A[ma + 1] if (ma + 1) < len(A) else float("infinity")
            Bleft = B[mb] if mb >=0 else float("-infinity")
            Bright = B[mb + 1] if (mb + 1) < len(B) else float("infinity")
            
            # condition for partition to be correct  
            if Aleft <= Bright and Bleft <= Aright:
                if total % 2 == 0:
                    return (min(Aright, Bright) + max(Aleft, Bleft)) / 2
                else:
                    return min(Aright, Bright)
            # modify l and r pointer of A
            elif Aleft > Bright:
                r = ma - 1
            else:
                l = ma + 1
```
