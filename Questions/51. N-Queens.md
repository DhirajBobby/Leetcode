
## Problem
#backtracking 
The **n-queens** puzzle is the problem of placing `n` queens on an `n x n` chessboard such that no two queens attack each other.

Given an integer `n`, return _all distinct solutions to the **n-queens puzzle**_. You may return the answer in **any order**.

Each solution contains a distinct board configuration of the n-queens' placement, where `'Q'` and `'.'` both indicate a queen and an empty space, respectively.

![[Pasted image 20260530234503.png]]

`Input: n = 4`
`Output: [[".Q..","...Q","Q...","..Q."],"..Q.","Q...","...Q",".Q.."]]`
`Explanation: There exist two distinct solutions to the 4-queens puzzle as shown above`

`Example 2:`

`Input: n = 1`
`Output: [["Q"]]`

## Solution
Since we know every row can have only 1 Q we call the backtracking function on every row.
Then for current row we check for every column if Q is placeable there. If no we check next column and so on till the end.
If yes, we place the Q and move to next row (That is, call backtrack with row +1)

To check if valid in constant time we maintain 3 sets. One for columns with Q, next for forward diagonals with Q and another for backward diagonals with Q. (row + col) and (row - col) represents all possible diagonals.

```python 
  class Solution:
    def solveNQueens(self, n: int) -> List[List[str]]:
        res = []
        board = [ ["."] * n for _ in range(n)]
        cols = set()
        diags1 = set()
        diags2 = set()

        def backtrack(row):
            if row == n:
                res.append(["".join(row) for row in board.copy()])
                return

            for col in range(n):
                # check if is valid in this position
                if not (col in cols or row + col in diags1 or row - col in diags2):
                    # place Q in this position and call backtrack for next row
                    board[row][col] = "Q"
                    cols.add(col)
                    diags1.add(row + col)
                    diags2.add(row - col)
                    # call for next row
                    backtrack(row + 1)
                    # Remove from this position for cleanup
                    board[row][col] = "."
                    cols.remove(col)
                    diags1.remove(row + col)
                    diags2.remove(row - col)
                    
        backtrack(0)
        return res
```
