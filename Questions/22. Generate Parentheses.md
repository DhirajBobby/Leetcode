
## Problem
#backtracking 
Given `n` pairs of parentheses, write a function to _generate all combinations of well-formed parentheses_.

`Example 1:`
`Input: n = 3`
`Output: []"((()))","(()())","(())()","()(())","()()()"]`

`Example 2:`
`Input: n = 1`
`Output: ["()"]`

## Solution
We use basic backtracking similar to subsets problem.
condition for branching are: can add "(" if total no. of "(" is less than n and can add ")" if total no. of ")" is less than total no. of "(".

If there are n open and n close parentheses it is a valid solution and we append it to res.

```python
  class Solution:
    def generateParenthesis(self, n: int) -> List[str]:
        res = []

        def backtrack(comb, op, cl):
            if op == cl and op == n:
                res.append(comb)
                return
  
            if op < n:
                backtrack(comb + "(", op + 1, cl)
            if cl < op:
                backtrack(comb + ")", op, cl + 1)

        backtrack("", 0, 0)
        return res
```
