## Problem
#array #queue #slidingwindow #heappriorityqueue #monotonicqueue 
You are given an array of integers nums, there is a sliding window of size k which is moving from the very left of the array to the very right. You can only see the k numbers in the window. Each time the sliding window moves right by one position.

Return the max sliding window.

`Example 1:`
`Input: nums = [1,3,-1,-3,5,3,6,7], k = 3`
`Output: [3,3,5,5,6,7]`
`Explanation:`
`Window position                Max`
`---------------               -----`
`[1  3  -1] -3  5  3  6  7       3`
`1 [3  -1  -3] 5  3  6  7       3`
`1  3 [-1  -3  5] 3  6  7       5`
`1  3  -1 [-3  5  3] 6  7       5`
`1  3  -1  -3 [5  3  6] 7       6`
`1  3  -1  -3  5 [3  6  7]      7`

`Example 2:`
`Input: nums = [1], k = 1`
`Output: [1]`

## Solution
We maintain a monotonic queue. (monotonic will be increasing or decreasing). 

We will maintain a decreasing monotonic queue. That is, we append an element if it is lesser than top of queue. The leftmost element will be the largest of that window. 

```python
 class Solution:
    def maxSlidingWindow(self, nums: List[int], k: int) -> List[int]:
        res = []
        deq = collections.deque()
        l = r = 0

        while r < len(nums):
            # pop as long as last element in queue is less than current element
            while deq and nums[r] > deq[-1]:
                deq.pop()
            deq.append(nums[r])
            
            # for initial k loops
            if (r + 1) >= k:
                res.append(deq[0])
                # When we increase l we could have taken largest out of queue.
                # Here we check this and update queue.
                if deq[0] == nums[l]:
                    deq.popleft()    
                l += 1
            r += 1 

        return res
```
