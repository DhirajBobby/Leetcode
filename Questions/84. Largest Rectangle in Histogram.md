## Problem
#array #stack #monotonicstack
Given an array of integers heights representing the histogram's bar height where the width of each bar is 1, return the area of the largest rectangle in the histogram.

`Example 1:`

![[Pasted image 20260609231316.png]]

`Input: heights = [2,1,5,6,2,3]`
`Output: 10`
`Explanation: The above is a histogram where width of each bar is 1.`
`The largest rectangle is shown in the red area, which has an area = 10 units.`

`Example 2:`
`Input: heights = [2,4]`
`Output: 4`

## Solution
For every element we try to find the max area possible including that element.
For this we need to know the how far we can so right and how far we can go left.
That is, for every element we need to find the first occurrence of an element smaller than it in both sides.

For this we can use a monotonic stack. (Similar to daily temperatures question).

We maintain two stacks - next small and previous small.
After creating the two stacks for every element we find the area and maximize it.   

```python
class Solution:
    def largestRectangleArea(self, heights: List[int]) -> int:
        l = len(heights)
        
        # To find next smaller element
        ns = [l] * l
        st = []
        for i in range(l):
            while st and heights[st[-1]] > heights[i]:
                ns[st.pop()] = i
            st.append(i)
            
        # To find previous smaller element
        ps = [-1] * l
        st= []
        for i in range(l-1,-1,-1):
            while st and heights[st[-1]] > heights[i]:
                ps[st.pop()] = i
            st.append(i)

        maxarea = 0

        for i in range(l):
            area = (ns[i] - ps[i] - 1) *  heights[i]
            maxarea = max(maxarea, area)
        
        return maxarea
        
```
